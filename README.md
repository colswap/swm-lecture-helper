# SWM Lecture Helper

SW마에스트로 멘토링/특강 게시판을 위한 Chrome 확장 프로그램.

700개 이상의 강연을 빠르게 검색하고, 비대면/대면 필터링하고, 즐겨찾기로 관리할 수 있습니다.

## 기능

- **전체 동기화** — 모든 페이지의 강연 데이터를 한 번에 캐싱
- **키워드 검색** — 강연 제목, 멘토명, 설명에서 검색
- **필터** — 날짜, 접수 상태(접수중/마감), 분류(멘토특강/자유멘토링), 장소(비대면/대면)
- **상세 수집** — 상세 페이지의 장소, 설명 등을 일괄 수집
- **즐겨찾기** — 관심 강연 저장 및 모아보기
- **상세 페이지 자동 캐싱** — 방문한 상세 페이지 정보 자동 저장

## 설치

1. 이 저장소를 다운로드하거나 클론합니다.
   ```
   git clone https://github.com/colswap/swm-lecture-helper.git
   ```
2. Chrome에서 `chrome://extensions` 열기
3. 우측 상단 **개발자 모드** ON
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 다운로드한 `swm-lecture-helper` 폴더 선택

## 사용법

1. [swmaestro.ai 멘토링 페이지](https://swmaestro.ai/sw/mypage/mentoLec/list.do?menuNo=200046)에 로그인
2. 확장 프로그램 아이콘 클릭 → **전체 동기화** (최초 1회, 이후 필요 시)
3. 검색창에 키워드 입력 또는 필터 조합으로 강연 탐색
4. **상세 수집** 버튼으로 장소(비대면/대면) 정보 추가 수집
5. ★ 버튼으로 즐겨찾기, "즐겨찾기" 탭에서 모아보기
