# SWM Lecture Helper

SW마에스트로 (swmaestro.ai) 멘토링/특강 게시판을 위한 Chrome 확장 프로그램.

700개 이상의 강연을 빠르게 검색하고, 비대면/대면으로 필터링하고, 즐겨찾기·신청 현황을 한 눈에 관리하며, 풀탭 **시간표**에서 빈 시간대 기반 검색까지 가능합니다.

## 기능

### 팝업 (빠른 조회)
- **빠른 검색** — 제목·멘토명·설명·장소 키워드
- **다중 필터** — 날짜, 접수 상태, 분류(멘토특강/자유멘토링), 장소(비대면/대면)
- **3개 탭** — 검색 / 즐겨찾기 / 내 신청
- **★ 즐겨찾기** — 관심 강연 저장
- **내 신청** — 내가 접수한 강연 한눈에 (과거 포함, 취소된 건 자동 제외)
- **장소 자동 수집** — 상세 페이지 정보(비대면: ZOOM 등)도 기본 동기화에 포함

### 시간표 (풀탭)
팝업 헤더의 **"시간표"** 버튼으로 풀탭 페이지 열기 (`chrome-extension://ID/timetable/timetable.html`)

- **주간 캘린더** — 7일 × 08:00~24:00 그리드, 신청 강연을 시간대별 블록으로 표시
- **사이드바 검색** — 팝업과 동일한 검색·필터·탭
- **호버 프리뷰** — 검색 결과에 마우스 올리면 그리드에 반투명 후보 블록 표시 + 충돌 경고
- **즐겨찾기 그리드 표시** — 헤더 토글로 즐겨찾기 강연을 노란 반투명으로 미리보기
- **상세검색 (시간대 선택)**:
  - 그리드 드래그 → 10분 단위 시간 범위 선택
  - 요일 헤더 클릭 → 그 요일 종일 토글
  - 우클릭 → 슬롯 개별 제거
  - **"내 빈 시간 전체"** 버튼 → 신청 강연 시간 빼고 빈 슬롯 자동 선택
  - 매칭: 강연 시간이 선택 슬롯에 *완전 포함* 되어야 결과
- **날짜 범위** — 시작·종료 두 입력으로 기간 지정
- **팝오버** — 블록 클릭 시 카테고리/시간/멘토/장소/설명 + ★ 토글 + "신청하러 가기" 버튼

### ⋯ 메뉴 (시간표 우상단)
- **iCal 내보내기** — 신청 강연을 `.ics` 로 다운로드 → Google/Apple 캘린더 import (Asia/Seoul 시간대)
- **인쇄 · PDF** — `@media print` 로 그리드만 깔끔하게 출력
- **데이터 초기화** — 모든 캐시·즐겨찾기·설정 삭제

### 백그라운드
- **자동 동기화** — 30분 주기 + 신규 강연 데스크톱 알림
- **빈자리 알림** — 즐겨찾기 강연이 만석에서 빈 자리 생기면 알림 (`count.current` diff 감지)
- **멘토 매치 알림** — 등록한 멘토 이름의 새 강연 등장 시 알림
- **알림 설정 UI** — 곧 추가 예정 (현재는 service-worker 기본 동작)

## 설치

### Chrome Web Store (심사 진행 중)

### 개발자 모드 (직접 로드)

```bash
git clone https://github.com/colswap/swm-lecture-helper.git
```
1. Chrome에서 `chrome://extensions` 열기
2. 우측 상단 **개발자 모드** ON
3. **압축해제된 확장 프로그램을 로드합니다** → `swm-lecture-helper` 폴더 선택

## 사용 흐름

1. [swmaestro.ai 멘토링 페이지](https://www.swmaestro.ai/sw/mypage/mentoLec/list.do?menuNo=200046)에 로그인
2. 확장 아이콘 → **접수중 동기화** (최초 1회, 이후 30분 자동)
3. 검색·필터·★ 즐겨찾기로 탐색
4. **시간표** 버튼으로 풀탭 열기 → 신청 강연 캘린더 + 빈 시간 기반 검색

## 키보드 단축키 (시간표)

| 키 | 동작 |
|---|---|
| `←` / `→` | 이전/다음 주 |
| `T` | 오늘 |
| `/` | 검색창 포커스 |
| `Esc` | 팝오버 닫기 |

## 개인정보 · 보안

모든 데이터는 브라우저의 `chrome.storage.local` 에만 저장되며 외부 서버로 전송되지 않습니다. 자세한 정책은 [PRIVACY.md](./PRIVACY.md).

## 기술 스택

- Chrome Extension Manifest V3
- Vanilla JS (프레임워크/번들러 없음)
- `chrome.storage.local`, `chrome.alarms`, `chrome.notifications`, `chrome.tabs`
- 모든 데이터 로컬 저장, 외부 라이브러리 없음
